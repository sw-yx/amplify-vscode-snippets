# amplify-vscode-snippets 

this is an unofficial snippets extension for AWS amplify

## Features

Snippets included:

- TypeScript/JavaScript
  - Amplify Configure
  - Authenticator
  - DataStore import/CRUD
- CSS
  - Root CSS
- GraphQL
  - @auth public read
  - @auth everything private
  - basic @model with no auth
